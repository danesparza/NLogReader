This is a single page browser-based application to access the LogReader service.

It's built using React (https://facebook.github.io/react/) and Flux (https://facebook.github.io/flux/)

For dependency management we use yarn/npm (https://docs.npmjs.com/getting-started/what-is-npm), webpack, and a package.json file in the root

To automagically detect changes and rebuild bundle.js, run 'webpack -w'

To make a production minified bundle.js build, run 'webpack -p'
