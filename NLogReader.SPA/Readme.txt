This is a single page browser-based application to access the LogReader service.

It's built using React (https://facebook.github.io/react/) and Flux (https://facebook.github.io/flux/)

For dependency management we use npm (https://docs.npmjs.com/getting-started/what-is-npm) and a package.json file in the root

To automagically detect changes and rebuild bundle.js, run 'npm start'

To make a production minified bundle.js build, run 'rpm run build'
